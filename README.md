# 🌌 Galaxy Wizard — Number Guessing Game

An interactive and galaxy-themed Number Guessing Game built using HTML, CSS, and JavaScript. The game includes multiple gameplay modes such as VS Computer, VS Friend, and Duel Mode with sound effects, responsive UI, and dynamic gameplay experience.

---

## 🚀 Features

### 🎮 VS Computer Mode
- Guess a randomly generated number within a dynamic range
- Limited attempts based on range difficulty
- Instant feedback for high/low guesses

### 🧙‍♂️ VS Friend Mode
- One player sets a secret number
- Another player tries to guess it
- Secret number visibility toggle option

### ⚔️ Duel Mode
- Two-player competitive guessing battle
- 5-round gameplay with score tracking
- Tie-breaker rounds for equal scores

### 🔊 Sound Effects
- Start game sound
- Correct guess sound
- Wrong guess sound
- Winning sound

### 📱 Responsive UI
- Galaxy-themed modern interface
- Smooth buttons and interactive design
- Mobile-friendly layout
